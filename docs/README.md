# 📖 Material Oficial del Curso

Este directorio contiene la documentación oficial de Google Cloud Cybersecurity, organizada y renombrada para facilitar su estudio secuencial.

---

### 📂 Módulos

* **`modulo_1_fundamentos/`**: Teoría de infraestructura de confianza, Chip Titan y modelos de servicio.
* **`modulo_4_redes/`**: Configuración de VPCs, reglas de firewall y segmentación lógica.
* **`casos_uso/`**: Escenarios prácticos (Cymbal Bank) para el análisis de incidentes.
