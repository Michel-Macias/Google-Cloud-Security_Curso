# 🛡️ Google Cloud Cybersecurity: Study Materials & Labs

Este repositorio contiene mi progreso, laboratorios y documentación consolidada durante la preparación para el **Google Cloud Cybersecurity Certificate**. Como profesional con perfil **SysAdmin Linux (ASIR)**, mi enfoque está en la transición hacia el análisis de seguridad en la nube, la automatización vía CLI y el endurecimiento (hardening) de infraestructuras globales.

---

## 🚀 Hoja de Ruta de Aprendizaje

El contenido está organizado de forma numérica para seguir el itinerario lógico de estudio:

1.  **`01_Plan_Estudio_Global.md`**: Control de fases y objetivos.
2.  **`02_Manual_Maestro_Analista.md`**: Guía rápida de conceptos clave.
3.  **`/docs/`**: Material oficial analizado y renombrado.
4.  **`/notes/`**: Resúmenes premium y cheatsheets de supervivencia.
5.  **`/labs/`**: Laboratorios prácticos ejecutados en sandbox.
